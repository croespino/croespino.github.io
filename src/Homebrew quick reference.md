- Verify health.
    ```
    brew doctor
    ```
- Search the `<PACKAGE>` formula.
    ```
    brew search <PACKAGE>
    ```
- Get `<PACKAGE>` information. 
    ```
    brew info <PACKAGE>
    ``` 
- Install `<PACKAGE>`
    ```
    brew install <PACKAGE>
    ``` 
- Upgrade `<PACKAGE>`
    ```
    brew upgrade <PACKAGE>
    ```
